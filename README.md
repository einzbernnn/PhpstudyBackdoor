# PhpstudyBackdoor
-u url目标
-t 线程数，默认10
-f 导入文件批量检测

python3 Phpstudybackdoor.py -u http://192.168.3.106
![Image text](https://github.com/einzbernnn/PhpstudyBackdoor/blob/main/20.png)

python3 Phpstudybackdoor.py -f url.txt


python3 Phpstudybackdoor.py -f url.txt -t 20
